Setting up the index file

index1.html

With this index file you can show a video and display a different before- and after the video section. In the HTML file you'll find two ID's: show-before and show-after.

Show before:


<!-- We're going to show this before the movie -->
<div id="show-before" class="text-white">
	<hr>


	<!-- Introduction -->
	<h5 class="bigtext serif italic margin-bottom">Hello!</h5>
	<h1 class="fittext uppercase bold">Let me tell<br>you a story</h1>


	<!-- Play button for the video -->
	<a class="play-btn" onclick="jQuery('#video').playYTP()">
		<i class="fa fa-play"></i>
	</a>


	<hr>
</div>

Show after:


<!-- We're going to show this after the movie and on mobile devices -->

	<h1 class="bigtext letterspace uppercase bold no-margin">Magnum</h1>

	<ul class="home-menu">
		<li>
			<a href="#work" class="smoothscroll">
				<i class="fa fa-briefcase"></i>
				<b>Work</b>
				<em>What I've Made</em>
			</a>
		</li>
		<li>
			<a href="#about" class="smoothscroll">
				<i class="fa fa-thumbs-o-up"></i>
				<b>About</b>
				<em>Personal Facts</em>

It's important to leave the names of these ID's as they are. There are no CSS styles attached to them, but the index.js file needs these ID's in order to work.

You can display anything you want inside the show-before and show-after ID's as long as you keep the (classname of the) play-btn as it is. Note that the show-after div contains the elements that are visible on mobile devices. At this time of writing YouTube doesn't support this function, so there is a fallback with a background image for these devices.

Also make sure to set the img-before and img-after that you'll find below the video controls in the HTML file.

If you want more information about setting up the video, read the instructions given in video settings.

index2.html

With this index file you can show a video and display for example your logo and the menu. Just like index1.html, your visitors are still in control of playing the video or not. Make sure to leave the classname play-btn-normal as is. There aren't any styles attached to it, but the index2.js file uses this classname in order for the videofunctions to work.

index3.html

This index file contains an autoplay and muted background video. If your visitor doesn't want to see the background video, there's a little pause button below the introduction screen. If you don't want the users to be in control of this button, simply delete it in your HTML file.


<!-- Video controls -->
<div class="video-controls">
	<button onclick="jQuery('#video').pauseYTP()"><i class="fa fa-pause"></i></button> <!-- Delete this line to prevent your users from pausing the video -->
	<button onclick="jQuery('#video').toggleVolume()"><i class="fa fa-volume-up"></i></button>
</div>

Of course the same goes for the volume button, however it's recommended that you don't start your website with a maximum volume video, because some people tend to forget that they left their speakers on at volume 1000. They probably want to shut off their browser - and your website - immediately. Result: visitor or client lost.

Video settings

Magnum supports YouTube background video's. Setting the background video is easy; simply copy the YouTube url from your browsers address bar and paste it in your HTML file. Example:


<!-- Background with youtube video -->
<div class="video-container">
	<a id="video" class="fullscreen-video" data-property="{videoURL:'http://www.youtube.com/watch?v=tDvBwPzJ7dY'}">The Video</a>
</div>
Settings

Below you'll find some settings for the background video plugin*. You can use these settings in the index[digit].js files.


opacity: 1              // (int) 0 > Set the opacity of the player;
mute: true              // (boolean) mute the audio;
showControls: true      // or false If you set the player as background you can show controls adding this parameter;
ratio: '4/3'            // or '16/9' to set the aspect ratio of the movie;
quality: 'default'      // or 'small', 'medium', 'large', 'hd720', 'hd1080', 'highres';
containment: 'body'     // (string) the CSS selector of the DOM element where you want the video background; if not specified it takes the 'body'; if set to 'self' the player will be instanced on that element;
optimizeDisplay: true   // (boolean) will fit the video size into the window size optimizing the view;
loop: true              // (boolean) or false loops the movie once ended.
vol: 100                // 1 to 100 (number) set the volume level of the video.
startAt: 20             // (int) Set the seconds the video should start at.
autoplay: true          // (boolean) or false play the video once ready.
showYTLogo: true        // (boolean) Show or hide the YT logo and the link to the original video URL.
showControls: true      // (boolean) Show or hide the controls bar at the bottom of the page.

* Source: YTPlayer documentation

Important!

At this time of writing YouTube doesn't support background video's on mobile devices. On smartphones and tablets Magnum shows a fallback background image. See also the fallback options at Choosing an index file.

High quality YouTube video's can be the reason of a browser crash on (older) computers that don't support high definition video's. If this is the case, it's recommended that you set quality to 'small' or 'medium'.

It's recommended that you don't use advertisements in your YouTube video's to prevent ad's of other companies on your website.

For more information about the plugin, visit the YTPlayer documentation.

How to setup the contact form

Adding your e-mailaddress

To recieve messages at your own e-mailaddress, go to line 60 in the contact.php file and add your e-mailaddress instead of the fake e-mailaddress. Example:


// Configuration option.
// Enter the email address that you want to emails to be sent to.
// Example $address = "hello@yourdomain.com";

$address = "your@email.com";
Changing the 'Thank You' message

To edit the 'Thank You' message that displays after someone sends a message, go to line 89 in contact.php and change the text between the <h4> and <p> tags. Example:


// Email has sent successfully, echo a success page.

echo "<fieldset>";
echo "<div id='success_page'>";
echo "<h4>Email Sent Successfully</h4>";
echo "<p>Thank you $name, your message has been submitted to us.</p>";
echo "</div>";
echo "</fieldset>";
Required fields

The name, e-mailaddress and comment/message fields are required. If you want the telephone field to be required, go to line 25 in contact.php and uncomment line 25 - 30.

To uncomment the code, simply delete the '//' symbols before the code.

Note: The contact form requires PHP and doesn't work locally. Please make sure to test your contact form on a server. Also make sure to test the contact form written by an unknown (fake) e-mailaddress. If these e-mails end up in your spam folder, please contact your webhost provider as this is most probably an issue with the server.

How to change the text

1. Open the HTML file with a text editor.
2. Edit the text between the <h1>, <p> and <em> (et cetera) tags, for example:


<h1>Magnum</h1>

becomes


<h1>Your name here</h1>

and


<h2>Graphic Design</h2>
<p>Lorem ipsum dolor sit amet.</p>

becomes


<h2>My Service</h2>
<p>Your text here.</p>

3. Save the file and view it in your browser to see the result.

How to change the font

1. Open the style.css file in the css folder with a text editor.
2. Search for the following text:


font-family: 'Open Sans', Helvetica, Arial, sans-serif;

3. Enter the fontname you wish to use. For example:


font-family: Georgia, Times, serif;

4. Save the file and view the result in your browser.

For more information about font families, visit w3schools.com.

Make sure to delete the @import code from the style.css file if you're not planning on using the Open Sans font. See also the Using Google Webfonts how to.

Using Google Webfonts

1. Visit https://www.google.com/fonts/.
2. Choose your favourite font, for example 'Droid Sans'.
3. Click the 'Quick use' button. (This is the grey button in the middle. You'll find it at the left of the blue 'Add to Collection' button.)

4. Select the styles of the font you wish to use, for example Normal 400 and Bold 700.
5. Copy the code you'll find at the thrid point and the second tab (the @import function) of the Google Webfonts page.

Example given:


@import url(http://fonts.googleapis.com/css?family=Droid+Sans:400,700);

6. Open your style.css file and add this code on the first line of the document.
7. Copy the font-family code given at point 4 of the Google Fonts instructions. Don't forget to add a fallback font after the Google Font:


font-family: 'Droid Sans', Helvetica, Arial, sans-serif;

8. Follow the instructions given in How to change the font.

Using the serif class

In the Magnum template you'll find a sans-serif and a serif font. The default is sans-serif, and if you'd like to display text in serif, simply use the serif class:


<p class="serif">Good stuff!</p>

Or apply the serif class to a container div:


<div class="twelve columns serif">
	<h1>Heading</h1>
	<p>Everything is serif in this div.</p>
</div>

You can also add the italic class to the serif class instead of using the <em> tags.

To change the serif class: Hit Command + F or Ctrl + F in the style.css file and look for "serif" so you can easily edit the font of the serif class.

Using the uppercase class

In the Magnum template you'll find some headings displayed in UPPERCASE. You can use the uppercase and no-uppercase class to easily switch styles. 


<p class="uppercase">Good stuff!</p>

You can also add the bold class to the uppercase class instead of using the <b> tags.

How to align text

By default almost every text on the website is centered. If you want to set a specific text to the left or right, simply add alignleft or alignright inside a row or column. For example:


<div class="twelve columns alignleft">
	<p>Everything in this div is aligned to the left.</p>
</div>

Fit text to the screen

You can use the fittext class to display text that will automatically scale down to the viewport:


<h1 class="fittext">This text is min 50px and max 100px and has a css fallback of 75px</h1>
<h2 class="fittext">This text is min 40px and max 80px and has a css fallback of 60px</h1>
<h3 class="fittext">This text is min 30px and max 60px and has a css fallback of 45px</h1>
<h4 class="fittext">This text is min 20px and max 40px and has a css fallback of 30px</h1>
<h5 class="fittext">This text is min 15px and max 30px and has a css fallback of 23px</h1>
<h6 class="fittext">This text is min 10px and max 20px and has a css fallback of 15px</h1>

For more information on using the FitText plugin, see the main.js file and also FitTextjs.com.

How to create a link

The tags used to produce links are the <a> and </a>. The <a> tells where the link should start and the </a> indicates where the link ends. Everything between these two will work as a link. The target of the link is added to the <a> tag using the href="http://www.domain.com" setting.

The example below shows how to make the word here work as a link to Domain.com.


Click <a href="http://www.domain.com">here</a> to go to Domain.

You simply:
1. Specify the target in the <a href=" ">.
2. Then add the text that should work as a link.
3. Finally add an </a> tag to indicate where the link ends.

Click here for more information about buttons.

How to link to a specific point in your file

For this example we're going to use the navigation menu of our website.
1. Go to the menu section in the HTML document
2. You'll see the following lines:


<a href="#home">Home</a>
<a href="#work">Work</a>
<a href="#about">About</a>
<a href="#services">Services</a>
<a href="#contact">Contact</a>

For example: The #services links to the services section. Let's take a look at where the content of that section starts:


<!-- Services -->
<section id="services" class="services">

3. The id of this section is 'services'. Whenever we change it to 'myservice', we only have to make sure to change the link in the menu as following:


<a href="#myservice">Any text you want</a>

You can do this for every item, however you don't have to change the classnames (class="services") because they're linked with the stylesheets. Changing those names will result in an unstyled section of your website.

How to create an icon

Magnum has the Font Awesome (fa) library of fonts inside this template. To place an icon of a home, simply place the following code in your HTML file:


<i class="fa fa-home"></i>

Making a link of it would result in the following code:


<a href="#home"><i class="fa fa-home"></i></a>

If you want to have more whitespace between an unstyled icon and text, use the inline class in the i class tag: 


<a href="#home"><i class="fa fa-home inline"></i></a>

To style the icon, you can choose a few pre defined styles: icon small, icon medium, icon and icon big. You can also use the icon huge class to display an icon as a huge graphic element in a column.

Code:


<a href="#home"><i class="fa fa-home icon small"></i></a>
<a href="#home"><i class="fa fa-home icon medium"></i></a>
<a href="#home"><i class="fa fa-home icon"></i></a>
<a href="#home"><i class="fa fa-home icon big"></i></a>

Result:

Styling the icon

To give the icons a specific look, you can add the following classes to it:

icon: this gives the icon a default dark background;
icon color, icon dark, icon light, icon white for icons with a specific background;
icon outline, icon outline color, icon outline dark, icon outline light, icon outline white for icons with a border.

If we want to have a big Facebook icon with a coloured border, this will result in the following code:


<a href="http://www.facebook.com" target="blank"><i class="fa fa-facebook icon big outline color"></i></a>

To view all available icons please visit: Font Awesome cheatsheet.

How to create a button

There are three different sizes of buttons in this template: a small sized button, a normal sized button and a large sized button:

Click me Click me Click me

Calling these buttons is simple:


<a href="#yourlink" class="button">Click me</a>

Small button:


<a href="#yourlink" class="button small">Click me</a>

Big button:


<a href="#yourlink" class="button big">Click me</a>
Styling the buttons

To give the button a specific style, you can use the same classnames as the icon styles*. For example:


<a href="#take-a-break" class="button outline dark">Click!</a>

Note that the font of the button is the default font of the style.css file.

Adding an icon inside a button

To display an icon inside a button, simply add the <i> between the <a> tags: 


<a href="#" class="button"><i class="fa fa-home"></i> Go home!</a>

How to choose the main color of your site

Magnum comes with different color stylesheets that you can use instead of the default color. Changing the main color of your website is very easy:
1. If you want to use your own color: Customize your color.css file
2. Open the index.html file (or any other .html file)
3. Look for the following lines in the head section:


<link rel="stylesheet" href="css/style.css">

5. Place the following line right below the style.css line - In the example below we're going to make the main color of the website orange:


<link rel="stylesheet" href="css/colors/orange.css">

6. Save the file and view the result.

How to customize the main color

1. If you'd like to choose your own color, copy and open any color.css file.
2. Simply replace your color in the color.css file:


/* Background color */
... {
background: #123ABC;
}

/* Color */
... {
color: #123ABC;
}

/* Border color */
... {
border: 2px solid #123ABC;
}

3. Make sure you call your color CSS file in the HTML documents below the 'style.css' line. (See also: How to choose the main color for your site.)

Note: You can use this tool or software such as Adobe Photoshop to choose a color.

How to use span colors

In the style.css file you'll find some span style colors to give your text specific colors. Adding these span colors is simple. Example given:


<p>This is a text with some <span class="text-color">coloured</span> text in it.</p>

You can choose between the following span classes:
- text-light
- text-white
- text-dark
- text-color

How to use text background colors

In the style.css file you'll find some text background colors to give your text specific background colors. Adding these background colors is simple. Example given:


<h1 class="text-bg>This is a text with a background color.</h1>

You can choose between the following span classes:
- text-bg - Displays a coloured background, same as text-bg.color
- text-bg light
- text-bg white
- text-bg dark
- text-bg color

It's recommended that you only use the text-bg class to headings in headers and or sliders.

How to set the background color of a section

In this example we're going to edit the background color of the 'social footer' section.

1. Open style.css in a text editor.
2. Search for 'social footer'. (Command + F, Ctrl + F) You'll see the following code:


.social-footer {
	/*...*/
	background: #f0f0f0;
}

3. Simply replace the '#f0f0f0' with your desired color.

Note: You can use this tool or software such as Adobe Photoshop to choose a color.

How to display an image

If you want to display an image on your website with for example dimensions of 1024x768 pixels, you can use the following code:


<img src="images/my-image.jpg" width="1024" height="768">

Magnum is packed with double sized images to provide a high quality look on devices with a higher pixel ratio, such as retina screens. There are several techniques on how to provide the best quality of images for your website on normal and high resolution screens.

Your logo

Below you'll find the code to display your logo in the introduction screen. It's recommended to also set the width and height* of your logo to appear on normal screens. In the example below the dimensions of the image are 180 x 140 pixels, so we're adding half the size in the <img> tag.


<img src="images/your-logo.png" alt="your-logo" width="90" height="70">

If you're using the fullscreen content and background slider (index 8 and 9), add the class preserve to your image to avoid a fullscreen logo:


<img src="images/your-logo.png" alt="your-logo" width="90" height="70" class="preserve">

Below you'll find the code for your logo in the menu. The max height of the logo is 30px for the best result. If you don't have a logo, you can also use a typographic logo between the <h1> tags. Make sure to uncomment the <!-- and --> codes.


<!-- Your symbolic or typographic logo here -->
<a href="#home" class="smoothscroll"><img src="images/your-menu-logo.png" alt="your-menu-logo" width="xxx" height="30" class="menu-logo"></a>
<!--<h1><a href="#home" class="smoothscroll">Your (Company) Name</a></h1>-->

* This goes only for images that are not inside a column. The images inside a column will automatically adapt to the width of a column if they're larger than the column's width itself.

Note about using a large logo

If you want to display a logo that is over 270px in width, you can use the the-logo class. This class sets the logo to a maximum of 600px and scales with the dimensions of the screen. You can change the width of this class in the style.css file. Hit 'ctrl + f' or 'cmnd + f' and look for 'the-logo'. Here you can set the max-width to the size you want to display your logo.

When you're using this class, you can delete the width and height tags in the HTML document, because the CSS class takes care of the dimensions now. Example given:


<img src="images/your-logo.png" alt="your-logo" class="the-logo">

And if you're using index 8 or index 9, you can combine this class with the preserve class:


<img src="images/your-logo.png" alt="your-logo" class="preserve the-logo">

How to configure a slider

Below you'll find some jquery codes that explain the working of the slider:


// Featured work slider
$('#featured-work-slider').bxSlider({
	auto: false, // You can slo set this to true
	mode: 'fade', // You can also set this to 'horizontal' or 'vertical'
	pager: true, // To display the bullets below the slider, if you don't want them, simply set this to false
	controls: true // Set this to false if you don't want to show the controls
});

If you're slider doesn't work, it's probably because of a missing ',' after a code. See the examples below:

Good:

// Featured work slider
$('#featured-work-slider').bxSlider({
	auto: false,
	mode: 'fade',
	pager: true,
	controls: true // The final code doesn't require a ','
});
Bad:

// Featured work slider
$('#featured-work-slider').bxSlider({
	auto: false,
	mode: 'fade' // Oops!
	pager: true,
	controls: true, // Oops!
});

Every slider needs its own ID. So if you want to show - for example - the Featured Work slider twice in your HTML file, make sure to copy the #featured-work-slider into its own ID: #featured-work-slider-two.

For more information on how to use the featured work & quote slider, please read the documentation of bxSlider.

How to use the grid

The grid is divided by twelve columns and based on the 960 grid system. For a normal use of the grid, you can create a column in a row as following:


<div class="row">
	<div class="four columns">
		<p>Your content here</p>
	</div>
</div>

This means that the area of the row is one-third of the entire width: 12 / 4 = 3. Example:

Another example: If you write six columns, this means the width of that area is half of the entire width of the document.

Offsets in the grid

If you want to offset a column, simply add 'offset-by-number' (any number between one and eleven) after the columns class. Example given:


<div class="row">
	<div class="ten columns offset-by-two">
		<p>Your content here</p>
	</div>
</div>

Centering the grid

If you want a column to be centered, simply add 'center' after the columns class. Example given:


<div class="row">

		<p>Your content here</p>
	</div>

Project page

Magnum is included with a special project page to show more details of your work. You can duplicate this HTML file as much as you want for every project. The ID of the slider can always remain the same, unless you want to display the slider multiple times on one page. If that's the case, make sure to give each featured-work-slider its own ID.

You'll notice the menu is slightly different than the menu on the main index files, because we have to link back to the index.html file. If you made adjustments in your main menu on the index.html file, and you want to copy the menu into your projectpage HTML file, make sure to link back to the index.html file and don't include the smoothscroll class.

404 page

Magnum is included with a special 404 page for your website's visitors who are lost (on your site). The 404 page has it's own CSS styles that you'll find in the style.css file.

There is no need for the 404 page to be indexed, so in your head tags, you can set the tag as follows:


<!-- Let's not index this page -->
<meta name="robots" content="noindex, nofollow">

Custom title settings

You can display a border below the titles of the sections or an icon behind the title. Make sure to add the <hr> that displays the border after the title and the <i> that displays the icon before the title.

Border:


<!-- The title with a border below -->
<div class="title">
	<h1>Hello Dolly</h1>
	<hr>
</div>

Icon:


<!-- The title with an icon behind -->
<div class="title">
	<i class="fa fa-bolt"></i>
	<h1>Hello Dolly</h1>
</div>

Transparent background

Because the body of certain index files (for example index3.html) have a class nobg (no background), the backgrounds of some parts are missing, so the background video or image is visible at those sections. If you don't want this, simply delete the class of the body:


<body class="nobg">

Becomes:


<body>

Menu colours

By default the main menu has a dark background. If you want the background to be white, add the white-menu class to it:


<!-- Menu -->
<div class="main-menu white-menu fixedmenu">

Footer styles

Magnum is included with 3 footer styles. The large social footer and the credits section are also working with the transparent background if the body class is set to nobg. See the different index[digit] files to choose the desired footer style and copy it into your index.html file.

Smoothscroll

When you click a link or button inside the page, you'll notice the page automatic scrolls to the target. If you want to create this effect, simply add the smoothscroll class to the anchor tag.


<p><a href="#contact" class="button smoothscroll">Get in touch</a></p>

When you apply the smoothscroll class to a link that can't be targeted in the same HTML file, the link doesn't work and the smoothscroll class needs to be removed.
